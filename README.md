# Learning Journal

A simple web prototype of a learning journal based on the course Learning How to Learn.

## Project Purpose

This tool helps students apply learning techniques in everyday study.

## Course Concepts Used

- Focused Mode
- Diffuse Mode
- Process over Product
- Recall
- Chunking
- Illusions of Competence
- Procrastination
- Motivation and Rewards

## How It Works

1. Set a learning goal.
2. Create a one-week learning contract.
3. Plan tomorrow before going to bed.
4. Complete a daily learning report.
5. Reflect on recall, chunks, and procrastination.
6. Use the anti-procrastination page when needed.

## Why This Helps

The journal turns course concepts into simple daily prompts. Instead of only reading about learning techniques, students use them while studying.

## How to Run

Open `index.html` in a browser.
